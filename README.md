# Dashboard em React com Tailwind CSS

Este repositório contém o código de um dashboard desenvolvido com **React** e estilizado com **Tailwind CSS**. O projeto foi criado para fornecer uma interface intuitiva e responsiva para exibir informações e dados de maneira prática e visualmente agradável.

## 🚀 Tecnologias e Ferramentas

- **React**: Biblioteca JavaScript para construção de interfaces de usuário.
- **Tailwind CSS**: Framework de CSS utilitário para estilização rápida e personalizada.
- **React Router**: Para navegação entre as páginas do dashboard
- **Linter e Prettier**: Ferramentas para manter o código consistente e limpo.

## 🛠 Funcionalidades

- **Painel Geral**: Exibe informações agregadas e indicadores principais.
- **Seção de Relatórios**: Visualiza dados em formato de tabela e gráfico.
- **Seção de Configurações**: Customiza o uso do dashboard.
- **Filtros Dinâmicos**: Filtra dados com base em parâmetros específicos.
- **Responsividade**: Interface totalmente adaptável a dispositivos móveis e desktops.

## 📦 Instalação

Para executar o projeto localmente, siga estas etapas:

1. **Instale as dependências**:
   ```bash
   npm install
   ```

2. **Inicie o servidor de desenvolvimento**:
   ```bash
   npm start
   ```

3. Acesse o dashboard em [http://localhost:3000](http://localhost:3000).
